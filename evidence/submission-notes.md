## Short Description

LocalVault AI is a local-first confidential document intelligence workspace for consumer laptops and desktops. It uses QVAC SDK to run local inference over private documents and produces summaries, citation-grounded answers, risks, action items, and auditable performance logs without cloud AI APIs.
