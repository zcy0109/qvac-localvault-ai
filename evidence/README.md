# Evidence Bundle

这个目录用于存放 QVAC Hackathon 评审所需的证据材料。

## 最终需要提交的材料

- `logs/latest-demo-run.json`：标准 demo 运行生成的最终推理日志。
- `api-disclosure.json`：项目使用的所有远程 API 调用披露。
- `hardware-screenshots/`：CPU、RAM、GPU、系统信息截图。
- `sample-documents/`：demo 视频中使用的本地样本文档。

注意：最终提交给 DoraHacks 的日志必须来自真实 QVAC provider，不能使用开发阶段的 fallback provider。
